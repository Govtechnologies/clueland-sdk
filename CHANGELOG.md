# Changelog

All notable changes to the Clueland SDK will be documented in this file.

## [1.0.0] - 2026-05-11

### Added
- Initial SDK release.
- **Engine Typings**: Comprehensive Lua annotations for IDE support.
- **JSON Schemas**: Validation for blocks and data types.
- **CLI Tool (`vox`)**: Support for `create` command using templates.
- **Mod Templates**: `basic_mod` for quick starts.
- **Example Mod**: `ruby_mod` demonstrating blocks and scripts.
- **Documentation**: Getting started guides and API reference.
