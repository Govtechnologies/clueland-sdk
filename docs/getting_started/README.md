# Getting Started with Clueland Modding

## Prerequisites
- **Visual Studio Code** (recommended)
- **Lua Language Server** extension for VS Code
- **Python 3** (for the `vox` CLI tool)

## Your First Mod
1. Navigate to the `templates` folder in the SDK.
2. Copy `basic_mod` to your own project folder.
3. Open the folder in VS Code.
4. Modify `scripts/init.lua` to add your logic.

## Using the CLI
The `vox` tool simplifies common tasks:
```bash
python sdk/tools/vox.py create my_mod
```
